Developing a Website

The first step to creating a website is to create a folder. Once the flder has been created create 3 folders inside of ot, These three folders are:

  • css
  • js
  • images

After the folders have been created open Sublime Text. Once Sublime has opened:

  1. Drag the website folder ontp the Subline Icon
  2. Create a new file - File --> New
  3. Save the file in the root of the website folder as index.html
